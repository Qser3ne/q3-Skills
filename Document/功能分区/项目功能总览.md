# 项目功能总览

## 项目目标

本项目用于集中维护 q3-Skills 技能集合。每个一级目录对应一个可独立加载或分发的 skill 模块，目录内通常包含 `SKILL.md`、可选的 `agents/openai.yaml`、脚本、参考资料或资源文件。

## 功能边界

- 负责保存技能定义、技能辅助脚本、参考文档与资源文件。
- 不负责提交本地运行缓存、解释器缓存或个人环境配置。
- 不在仓库内保存真实 token、Cookie、密码或私钥。

## 当前状态

- 已初始化本地 Git 仓库。
- 已配置远程仓库：`git@github.com:Qser3ne/q3-Skills.git`。
- 已添加 `.gitignore`，排除 Python 缓存和常见系统临时文件。

## 验收方式

- `git status --short --branch` 显示工作区状态清晰。
- `git remote -v` 显示 origin 指向目标 GitHub 仓库。
- 首次提交后可通过 `git push -u origin main` 推送到远程仓库。
