# 仓库管理

## 仓库信息

- 本地路径：`/home/qser3ne/PrjMD/q3-Skills`
- 默认分支：`main`
- 远程仓库：`git@github.com:Qser3ne/q3-Skills.git`

## 提交范围约定

- 提交 skill 源文件、说明文档、脚本、参考资料和必要资源。
- 不提交 `__pycache__/`、`*.pyc`、系统临时文件或个人环境文件。
- 涉及凭据的 skill 只能记录环境变量或参数约定，不得写入真实凭据。

## 常用命令

- 查看状态：`git status --short --branch`
- 查看远程：`git remote -v`
- 暂存改动：`git add <path>`
- 提交改动：`git commit -m "<message>"`
- 推送主分支：`git push -u origin main`

## 本轮续接上下文

- 本轮任务是创建本地 Git 仓库、绑定远程仓库、创建首次提交并推送。
- 提交前需要确认 `.gitignore` 生效，避免 Python 缓存进入首次提交。
- 如远程仓库非空，推送可能被拒绝；需要先检查远程历史后再决定合并或改用其他策略。
